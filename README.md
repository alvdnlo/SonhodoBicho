## ⚠️ Aviso Importante

**Este projeto é 100% entretenimento.** Não tem nenhum compromisso com religião, misticismo, espiritualidade ou qualquer crença. O "Pai Véio do Bicho" é um personagem fictício e cômico inspirado na cultura popular brasileira — nada aqui deve ser levado a sério. Os números gerados são aleatórios e nenhum resultado garante ganhos em jogos ou apostas.

O jogo do bicho é ilegal no Brasil. Jogue com responsabilidade.

---

## 💡 Por que esse projeto existe?

Esse projeto nasceu de uma pergunta simples: **como usar IA generativa para criar experiências criativas e divertidas?**

A ideia foi transformar uma tradição popular brasileira — o Livro dos Sonhos e o Jogo do Bicho — em uma aplicação web com personalidade, humor e identidade visual forte. O objetivo não foi criar algo útil no sentido tradicional, mas demonstrar que:

- É possível construir **produtos com personalidade** usando apenas HTML, CSS e JS
- **Prompt engineering** bem feito transforma uma IA genérica em um personagem único
- **Design com identidade cultural** diferencia um projeto de portfólio comum
- Integração com **APIs de IA gratuitas** abre possibilidades reais sem custo

---

## 🛠️ Tecnologias Usadas

- **HTML, CSS e JavaScript** puro — sem frameworks, sem dependências
- **Groq API** com modelo `llama-3.3-70b-versatile` — inferência rápida e gratuita
- **Google Fonts** — Abril Fatface, Kalam, Oswald
- Deploy via **GitHub Pages**

---

## 🎨 Decisões de Design

O visual foi pensado para fugir do padrão genérico de projetos com IA. A estética mistura:

- Cores quentes de **festa junina** com o clima noturno do **subúrbio carioca**
- Tipografia expressiva que remete a **cordel e cartazes populares**
- Bandeirolas animadas, brilhos e microanimações para dar vida à página
- Fundo escuro com acentos em dourado e verde — longe do clássico branco com gradiente roxo

---

## 🤖 Prompt Engineering

O coração do projeto é o prompt do "Pai Véio do Bicho" — um personagem construído com:

- **Persona definida**: Bezerra da Silva + Chacrinha + Mussum
- **Tom cômico obrigatório**: a IA é instruída a zoar o sonho antes de revelar os números
- **Base de conhecimento embutida**: tabela completa dos 25 bichos e símbolos do Livro dos Sonhos
- **Saída estruturada em JSON**: garante que todos os campos do resultado sejam preenchidos corretamente
